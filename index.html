CI/CD with Jenkins from Git to Private Docker Registry

This is a simple but very attractive project. We have a python-flask web application running as a docker container and whenever we push changes to our git repository, our Jenkins creates and pushes (or updates) a docker image to our local docker-registry and deploys it to our host. As a result, we have 4 containers: docker-registry, portainer(for docker monitoring), jenkins and python-flask. All within the same docker network.
